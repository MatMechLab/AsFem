AsFem

This is A Simple Finite Element Method program, which is short for AsFem, for the phase-field modeling and multiphysics simulation.

The package is written in C++ and heavily depends on the PETSc library for efficient computing and simulation. It is mainly designed for solid mechanics problems and phase-field modeling.
